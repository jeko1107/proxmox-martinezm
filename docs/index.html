Proxmox Martinez — Sitio público

Esta es una página de ejemplo publicada desde la carpeta docs/ mediante GitHub Pages.

¿Qué sigue?

  1. Reemplaza los archivos en docs/ con tu sitio estático (HTML/CSS/JS).
  2. Haz push a la rama main. El workflow .github/workflows/deploy-pages.yml publicará automáticamente el contenido.
  3. En la configuración del repositorio, en Pages, selecciona la opción para servir desde GitHub Actions (debería activarse automáticamente al primer despliegue).

Comandos útiles

git add docs/*
git commit -m "Actualiza sitio público"
git push origin main
